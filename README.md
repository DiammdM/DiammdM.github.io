# diammdm.github.io

Personal website built with Next.js (static export) and deployed to GitHub Pages.

## Local dev

```bash
npm install
npm run dev
```

## Deploy

Pushing to `main` triggers the GitHub Actions workflow: **Deploy to GitHub Pages**.

## Resume

Add your resume PDF here:

- `public/resume.pdf`

Then the site will serve it at:

- `https://diammdm.github.io/resume.pdf`

## SEO

- OpenGraph image: `public/og.svg`
- Sitemap: `/sitemap.xml`
- Robots: `/robots.txt`
